open(method, path, asynchronous, usernameopt, passwordopt)

Source:
  • xhr.js, line 38
Se encarga de realizar la apertura a la invocación del recurso.
Parameters:
Name Type Attributes Description
method String Es el tipo de petición que se realizará. Su valor puede ser: "CONNECT", "DELETE", "GET", "HEAD", "PATCH", "POST", "PUT", "OPTION" y "TRACE".
path String Hace referencia a la ruta o destino (url) del recurso a consultar.
asynchronous Boolean Señala si la petición debe ser síncrona (true) o asíncrona (false).
username String <optional>
 Opcional, en caso de ser necesario, establece el nombre de usuario. Su valor, sino se establece, es "NULL".
password String <optional>
 Opcional, en caso de ser necesario, establece la contraseña. Su valor, sino se establece, es "NULL".
